## 验收标准（技术视角）

- [ ] **分类逻辑**:
  - [ ] 优先使用本地静态白名单（识别 Reddit, CNN, GitHub 等）。
  - [ ] 针对未知域名，调用 LLM 进行特征识别分类（Media, Blog, Social, Corporate, Academic, Forum, Other）。
- [ ] **性能优化**:
  - [ ] 实现简单的 LRU 缓存或数据库缓存，避免重复分类同一域名。
- [ ] **数据提取**:
  - [ ] 能够从海量结果中高效提取域名并进行分类统计。
