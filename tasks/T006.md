## 验收标准（技术视角）

- [ ] 中间件 (Middleware) 功能：
  - [ ] 提取 `subdomain` 或特定 Header (`x-tenant-id`)
  - [ ] 验证 Supabase JWT 令牌 (SSR)
  - [ ] 若未登录重定向至 `/login`
- [ ] 客户端状态 (Client Context)：
  - [ ] 提供全局 `useTenant()` hook 获取当前 `tenant_id`
  - [ ] 提供 `useAuth()` hook 获取当前用户信息
- [ ] 服务器端 (Server-side)：
  - [ ] `getTenantId()` 能在 Server Component 和 Route Handlers 中安全地通过 Header 或 Cookies 提取 `tenant_id`
- [ ] 确保在所有 API 调用和数据库查询中强制注入 `tenant_id`
